Summary
-------
This agent helps convert the project from static dummy data to a Supabase-backed
production application: auditing sources of dummy data, replacing them with
queries, implementing secure admin CRUD APIs, wiring uploads, and preparing
environment, CI/CD and deployment instructions.

Suggested first prompt examples
- Audit repo for dummy data and Supabase usage.
- Replace `src/data/events.ts` with a server-side query and update the UI.
- Implement `src/app/api/v1/events` endpoints with auth checks.

Ambiguities to clarify before making major edits
- Should data fetching prefer Next.js server components (recommended) or client
  components for real-time interactions?
- Which deployment target do you prefer (Vercel recommended for Next.js)?

End of agent definition.
