# APEX Hub Website

This repository contains the source code for the **APEX Hub Website** — a unified digital portal for the **APEX Startup Group (ASG)** and the **APEX AI Launchpad (AAL)** based in Jalgaon, North Maharashtra.

## 🚀 Quick Start Guide

### 1. Installation
Clone the repository and install the project dependencies:
```bash
npm install
```

### 2. Running Locally
Launch the local Vite development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.

### 3. Production Build
Compile and optimize the project for production deployment:
```bash
npm run build
```

## 🛠️ Tech Stack Highlights
- **React 19 & Vite 8** for rapid execution and compilation.
- **Tailwind CSS 3** custom design tokens paired with clean CSS transitions.
- **Anime.js 4** for synchronized layout entry animations.
- **React Router DOM 7** handling clean clientside routes.
- **Radix UI** primitives for high accessibility and state-driven menus.
